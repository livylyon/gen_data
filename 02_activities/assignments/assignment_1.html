Question 2: Allele Frequency Estimation

  1. Load the genotype matrix for SNPs rs1861, rs3813199, rs3128342, and rs11804831 using additive coding. What are the allele frequencies (AFs) for these four SNPs?
# Create a text file with four target SNPs
printf "%s\n" rs1861 rs3813199 rs3128342 rs11804831 > 02_activities/data/q2_snplist.txt

# Use PLINK to extract them and create an additive coded .raw file
plink2 --bfile 02_activities/data/gwa.qc.A1 \
      --extract 02_activities/data/q2_snplist.txt \
      --recode A \
      --out 02_activities/data/q2_geno_additive
# Load the additive genotype matrix 
geno_q2 <- fread("02_activities/data/q2_geno_additive.raw")

# Calculate Allele Frequencies (AFs)
allele_freq <- apply(geno_q2[, 7:ncol(geno_q2)], 2, function(x) {
  sum(x, na.rm = TRUE) / (2 * sum(!is.na(x)))
})
